# CS2 Profile Stats
Extension that allows you to see player's statistics from Steam, Leetify and FaceIT right in their Steam profile.  
Comes for firefox and chrome, while also possible to install in your steam client without any plugins.

## Links
Firefox: https://addons.mozilla.org/en-US/firefox/addon/cs2-profile-stats  
Chrome: https://chromewebstore.google.com/detail/cs2-profile-stats/cdjdnohbfomlobiemcceejdnobfgnpje

## Steam installation
- Open any link in a new tab in your Steam client.
- Enter `https://chromewebstore.google.com/detail/cs2-profile-stats/cdjdnohbfomlobiemcceejdnobfgnpje` and press enter.
- Click `Add to Chrome`, you will be prompted to save the extension file.
- Save the file and the extension is now installed.

## Manual installation
Just follow steps in [cs2-profilestats-extension](https://github.com/CS2-Profile-Stats/cs2-profilestats-extension) and [cs2-profilestats-api](https://github.com/CS2-Profile-Stats/cs2-profilestats-api).
